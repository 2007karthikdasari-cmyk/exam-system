# Exam System - Online Examination Platform

A complete online examination system with timer, admin panel, student analytics, and email notifications.

## Features

### Admin Features
- Create and manage exams
- Add questions with image support
- View student performance analytics
- Send scores via email
- Question-wise analysis
- Student management

### Student Features
- Register and login
- Take timed exams
- Randomized questions per student
- Auto-submit on time completion
- View performance analytics
- Exam history with detailed results

## Tech Stack

- **Backend**: Node.js, Express.js
- **Database**: PostgreSQL
- **Frontend**: EJS, Bootstrap 5, jQuery
- **Authentication**: bcryptjs, express-session
- **Email**: Nodemailer
- **Deployment**: Render
